# discordjsx

A fork of [discord-jsx-renderer](https://github.com/deniz-blue/discordjsx) with additional features for SimplyPlural Integration.

## Installation
Simply install react and this package from the git repo:
`npm install react github:SimplyPlural-Discord-Integration/discordjsx`

## Usage
Upstream docs are maintained [here](https://djsx.deniz.blue/). Additions to the API are documented on the [wiki](https://github.com/SimplyPlural-Discord-Integration/discordjsx/wiki).

## Contributing
While contributions are welcome, contributions should be directed [upstream](https://github.com/deniz-blue/discordjsx) unless it makes use of changes specific to this fork or our repos making use of this fork.
